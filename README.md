# TRC_1_3

git status — перевірити, чи немає забутих файлів (вони будуть червоними).

git add . — додати всі зміни до наступного збереження.

git commit -m "Add StudentService logic and Unit Tests" — зафіксувати зміни з описом.

git push origin main — відправити все на GitHub.

git branch - перевірка

git checkout -b feature/student-crud - створити та перемкнутись на гілку 

git checkout main - повернутись на головну гілку 

git merge feature/student-crud - витягнути всі змімни з робочої гілки main

git push origin main - відправити оновлений код на GitHub
